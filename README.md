# Child Online Safety Text Classifier

A BERT-based classifier to detect toxic/harmful content in text, supporting child digital safety.

## Purpose
Demonstrates AI engineering skills for protecting children online (relevant to roles at companies like Qoria/Linewize).

## Features
- Multi-label toxicity detection
- Real-time inference
- Clean, user-friendly interface

## Tech Stack
- Python, PyTorch, Transformers (BERT)
- Streamlit for deployment

Live Demo: [Streamlit App](https://harmguard-ai-fine-tuned-bert-for-online-child-safety-dbtj68xcb.streamlit.app/)
